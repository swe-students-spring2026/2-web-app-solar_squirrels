## Steps necessary to run the software

Clone the repository, then copy the contents of the provided .env file into the "main" folder within the repository. From the repository root, run `pip install pipenv` if pipenv is not already installed, then `pipenv install --ignore-pipfile` to install all dependencies. Run `pipenv shell` to activate the virtual environment, then `pipenv run python main/run.py` will start the Flask server. Navigate to http://127.0.0.1:5001/ in the browser. You may then create an account, or use the provisional credentials (user: test password: test) to log in and begin using the app.
